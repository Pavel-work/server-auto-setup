# Server Auto Setup

Одна команда для полной установки Docker, PostgreSQL, Qdrant, Ollama, Apache, Nginx Proxy Manager, Portainer, Supabase и n8n на сервере с Ubuntu.

## Быстрый запуск

Скопируйте и выполните в терминале вашего сервера (Ubuntu) одну строку:


```bash
curl -fsSL "https://raw.githubusercontent.com/Pavel-work/server-auto-setup/main/install.sh" | sudo bash -s
```

**Джентльменский набор** — это мощный интерактивный bash-скрипт для быстрого развертывания популярного серверного стека на базе Docker. Скрипт берет на себя всю рутину: от установки Docker и настройки сетей до генерации `docker-compose.yml` и инициализации баз данных.

## ✨ Ключевые особенности

- 🧭 **Интерактивное TUI-меню**: Навигация "Назад/Далее", выбор только нужных сервисов.
- 💾 **Сохранение состояния**: Можно прервать установку и продолжить позже. Все пароли, порты и API-ключи сохраняются в зашифрованном виде.
- 🐳 **Умная работа с Docker**: Автоматическая установка Docker и **самопочинка** `containerd` при возникновении ошибок `failed to lease`.
- ⚙️ **Динамический `docker-compose.yml`**: Файл генерируется "на лету" строго под выбранные вами сервисы, без мусора.
- 🗄️ **Поэтапный запуск БД (v4.11)**: Умная инициализация. Если выбраны n8n и PostgreSQL, скрипт сначала поднимет БД, дождется ее готовности и создаст базу для n8n через `superuser` с ретраями, и только потом запустит остальные контейнеры.
- 🛡️ **Проверка портов**: Скрипт сам проверит, заняты ли порты, и предложит выбрать альтернативу.
- 🛠️ **Post-install меню**: Скрипт не "исчезает" после установки. Вы получаете панель управления для добавления новых сервисов, переустановки, просмотра логов или полного сброса.

## 📦 Включенные сервисы

| Сервис | Описание |
| :--- | :--- |
| **PostgreSQL** | Надежная реляционная БД (с авто-инициализацией и созданием БД для n8n). |
| **Supabase** | Open-source альтернатива Firebase (Auth, DB, Storage, Realtime). |
| **n8n** | Платформа для автоматизации рабочих процессов (no-code/low-code). |
| **Ollama** | Локальный запуск LLM (Llama 3, Mistral и др.). |
| **Qdrant** | Высокопроизводительная векторная БД для AI и RAG. |
| **Nginx Proxy Mgr** | Удобное управление реверс-прокси и бесплатными SSL-сертификатами. |
| **Portainer** | Визуальное управление Docker-контейнерами. |
| **Apache** | Классический веб-сервер для хостинга статики/сайтов. |

## ⚙️ Системные требования

- **ОС:** Ubuntu / Debian
- **Права:** `root` или `sudo`
- **Сеть:** Активное интернет-соединение
- **Диск:** Минимум 5 GB свободного места
- **Пакеты:** `dialog` (установится автоматически при первом запуске)

## 🚀 Быстрый старт

Скачайте и запустите скрипт одной командой:

```bash
# Если скрипт загружен на сервер:
sudo bash install.sh

# Или одной строкой (замените ссылку на raw-ссылку вашего репозитория):
curl -sSL https://raw.githubusercontent.com/YOUR_USER/YOUR_REPO/main/install.sh | sudo bash
```

## 🖥️ Как это работает

1. **Приветствие и выбор:** Отмечаете галочками (Пробел) нужные сервисы.
2. **Настройка:** Скрипт спросит параметры (пароли, порты, домены, LLM-провайдеры) *только* для выбранных сервисов.
3. **Установка:** Автоматическая установка Docker (если не установлен), генерация `.env` и `docker-compose.yml`.
4. **Запуск:** Поэтапный старт контейнеров (сначала БД, затем зависимые сервисы).
5. **Финал:** Вывод сводки с IP-адресами, портами и сгенерированными паролями.

## 🛠️ Post-Install Меню

После завершения установки скрипт предлагает меню управления:
- ➕ **Добавить сервисы** к уже работающему стеку.
- 🔄 **Переустановить** конкретный сервис (с сохранением или удалением данных).
- 📊 **Статус** контейнеров.
- 📄 **Логи** установки (`/var/log/install.log`).
- 🗑️ **Полный сброс** (удаление всех контейнеров, томов, сетей и настроек).

## ⚠️ Дисклеймер

Скрипт предназначен для быстрого развертывания dev/test окружения, домашних серверов (HomeLab) или небольших production-серверов. 
**Всегда делайте бэкапы** перед использованием функций переустановки или полного сброса!
